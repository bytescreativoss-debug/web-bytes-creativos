## 🤖 Chatbot y Configuración
El sistema ya cuenta con un asistente inteligente para ventas.

* **Importante**: El archivo `.env` está incluido con la `OPENAI_API_KEY` para asegurar que el bot responda desde el primer segundo.
* **Privacidad**: Mantener este repositorio siempre como **PRIVADO** para proteger las credenciales.

## 🚀 Guía de Inicio Rápido
1. **Clonar** el repo:
```
git clone https://github.com/Criscorre/web-bytes-creativos.git
```

2. **Instalar dependencias**:
```
npm install
```

3. **Encender Backend**:
```
node server.js
```

4. **Encender Frontend**:
```
npm start
```
