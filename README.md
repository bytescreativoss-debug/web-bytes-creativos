Bytes Creativos - Ecosistema Web 2026

Este repositorio contiene la infraestructura oficial de **Bytes Creativos**, optimizada para el mercado de E-commerce estratégico.

## 🛠️ Tecnologías
* **Frontend**: React.js + Tailwind CSS (Diseño Cyberpunk).
* **Backend**: Node.js + Express (Puerto 5000).
* **IA**: Integración con OpenAI (GPT-3.5 Turbo).

## 🤖 Chatbot y Configuración
El sistema ya cuenta con un asistente inteligente para ventas. 
* **Importante**: El archivo `.env` está incluido con la `OPENAI_API_KEY` para asegurar que el bot responda desde el primer segundo. 
* **Privacidad**: Mantener este repositorio siempre como **PRIVADO** para proteger las credenciales.

## 🚀 Guía de Inicio Rápido
1. **Clonar** el repo: `git clone https://github.com/Criscorre/web-bytes-creativos.git`
2. **Instalar dependencias**: `npm install` (y en la carpeta del servidor si aplica).
3. **Encender Backend**: `node server.js`
4. **Encender Frontend**: `npm start`
